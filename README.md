# Magalir Auto 

Magalir Auto is a project created with the idea of improving safety and convenience for women commuters. This application focuses on providing a reliable and secure auto booking experience designed specifically for women.

## Project Overview
The main goal of this project is to address real-world safety concerns faced by women while using public transportation. It provides a simple and accessible solution that makes booking autos easier and safer.
